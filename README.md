# Jianshu Sina Crawl

## Jianshu
Get the recommended authors of Jianshu and make them into word clouds. Publish articles through Sina Weibo. The article information includes author's nickname, number of followers, number of articles published, word number, diamond number of Jianshu, author's head image, and word clouds of recent articles.

## Sina Weibo
Simulate login on Sina Weibo, upload word clouds through image upload interface, and finally publish combined information. In the process of simulated login, encounter the verification code, send it to the Wechat file assistant through itchat, receive the verification code and reply to the background processing, and push the message to the Wechat file assistant after publishing success.
